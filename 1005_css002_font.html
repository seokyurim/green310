font 속성

글꼴, 글자크기, 글자굵기, 기울임, 줄간격, 대소문자, 글자색상 등 7가지로 정리

색상을 제외한 여섯가지 속성은 한번에 작성가능
속성 풀이
font -family 글꼴을 적용하는 속성으로 ''(따옴표)로 표기, serif(꺾임체)와 sans-serif(비꺾임체)로 구분함
font-size 글씨의 크기를 단위와 함께 작성 (px,pt,%,em등으로 작성)
large,small과 같이 기준단위보다 한단계 크고 작게도 가능
font-wight bold,normal로 표기/ 숫자표기시 600/400이 기준
font-style litalic,normal로 표기/ 기울임체
font-variant small-caps, normal로 표기/ 작은대문자로 표기
line-height 행간을 나타내고 단위가 1 → 100% 기본값이 1.6임
font 위의 내용을 한줄에 표기가능
font {1번칸 2번칸 3번칸}
  • 1번칸: font-weight, font-style, font-variant
  • 2번칸: font-size/ line-height 순서바뀔 수 없음 line-height는 생략 가능
  • 3번칸: font-family
1번칸은 생략이 가능하나. 2번칸과 3번칸은 빠지면 안됨 순서가 123은 바뀌면 안됨

웹폰트 활용법: 구글폰트에서 폰트를 선택 후 링크 혹은 import를 먼저 복사해서 기입 후 font-family를 복사해서 적용

color

color는 글자의 색을 나타내며 background, border, shadow 등과 같이 컬러를 사용하는 공간에서 표현방식은 모두 같다.

  • 이름 지정 : red, white, black
  • 핵사코드지정 : #000000(검정), #FFFFFF(흰색) 단 코드 이름이 중복될 경우 3자리로도 가능함 #000, #FFF
  • RGB숫자지정 : RGB(0,0,0)검정 RGB(255,255,255)흰색
  • RGBA 숫자지정 : A - 투명도를 나타내는 ALPHA 값으로 0부터1까지 소수점표기 0(투명)1(불투명)
  • opacity : 투명도 지정한 모든 요소의 내부내용이 투명해짐
