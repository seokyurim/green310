html의 시작

시맨틱태그란?

의미를 부여받은 태그로 특별한 성질을 가지고 있는 태그로 사용된다. 쓰임새에 맞게 바르게 사용되어야한다.

헤더 - header

주로 맨위쪽이나 왼쪽에 검색창이나 사이트 메뉴와 로고들을 삽입하여 사용한다. 머릿말에 해당하는 문장으로 footer 태그 내부에 삽입할 수 없다.

네비게이션 - nav

웹문서를 다른문서 혹은 다른사이트랑 연결을 하는데 사용된다. 어느 태그내부에도 사용가능하고 또한 단독으로도 사용가능하다.

본문 - main

웹문서에 핵심이 되는 영역으로 본문영역이라고 할 수 있으며 하나의 html안에 1번만 사용가능하다.

콘텐츠 영역 - article

내용을 담고 있는 소규모의 영역, 기사, 짧은 글 등으로 표현, 여러개 사용가능하며 article안에 또 다른 article를 담을 수도 있음

콘텐츠 블록 영역 - section

영역을 모아두는 콘텐츠영역, article를 담아서 그룹핑할 때 편리, 그런데 section 또한 article 내부에도 넣을 수 있음

본문 외 영역 - aside

main 영역 외에 왼쪽, 오른쪽, 아래쪽 등 본문내용 외의 내용을 담을 때 사용

푸터 - footer

모든 태그가 다 들어가도 큰 문제는 없으나, 웹접근성과 웹표준에는 위반 될 수 있음

제목 태그란?

heading의 약자로 h와 숫자 1~6까지의 조합으로 생성 숫자가 커질 수록 크기가 작아짐

  • 특징1 - 볼드체로 나타남
  • 특징2 - 글씨의 크기가 일반보다 크다가 작아짐
  • 특징3 - 위아래 여백을 가지고 자동줄바꿈 됨

단락 태그란?

paragraph의 약자로 p태그로 사용하며 텍스트단락을 사용할 때 가장 간편하게 사용

  • 특징1 - 내부안의 글씨는 화면크기에 자동으로 맞추고 내용이 끝날 때 자동줄바꿈이 됨
  • 특징2 - 위아래 여백을 가지고 있음
  • 특징3 - hn 태그는 p 태크 내부에 사용 불가, hn 태그안에도 쓸 수 없고, p태크 안에도 쓸 수 없음

줄바꿈 태그란?

강제 줄을 바꾸기위해 enter의 기능을 하듯이 문장을 아래로 바꿀 수 있음
break의 약자 br 태그라고 사용

  • 특징1 - 짝궁이 없음
    혼자만 단독으로 사용

박스 태그란?

아무런 특징이 없는 태그로 어느곳에서든지 자유롭게 사용이 가능

  • 블럭태그 - div(시맨틱태크의 모체) 자동줄바꿈 성질만 가지고 있음
  • 인라인태그 - span (자동줄바꿈이 없어 어느곳이든 묶음이 필요하면 사용가능)

텍스트 꾸밈 태그란?

인라인태그의 성질로 다른태그 내부에 사용이 되면서 글씨의 특징을 바꾸는 태그들

  • 볼드체 - b, strong strong
  • 이탤릭체 - i i , em em
  • 취소선 - del
  • 밑줄 - ins

하이퍼링크란?

앵커태그인 a태그로 사용되며, href라는 속성을 사용하여 링크가 위치할 주소를 작성해준다.
속성은 여러개 사용가능하며 같은 속셩명은 여러번 쓸 수 없다.

  • 특징1 - 색이 바뀌고 밑줄이 생김
  • 특징2 - 커서 모양이 바뀜
  • 속성 href="링크주소"
  • 속성 targat="_blank" 새창열림

이미지태그란?

img 라는 태그로 사용되며 src="이미지경로주소"라는 속성을 필수적으로 사용하며,
경로가 바르지 않을 경우 사진이 정상적으로 나타나지 않음

  • 속성 src="경로주소"
  • 속성 alt="대체용 텍스트" 웹접근성을 고려하여화면 낭독기를 통해 불려지는 텍스트
